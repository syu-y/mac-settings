# Starship

Promptの装飾用プラグインStarship
採用理由 = 流行ってた & 見やすかった

## 導入

[公式](https://starship.rs/)がドキュメントの日本語対応してくれているので手順通りに導入。

1点だけ注意としてzshの場合、`.zshrc`に設定追加となっているがPreztoの場合は.zshrcがエイリアスでリンク先の`zshrc`に設定を追加する。
